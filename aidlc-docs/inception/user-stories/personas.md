# ペルソナ定義

## ペルソナ1: 田中 陽介（テックブロガー）

| 項目 | 内容 |
|---|---|
| **名前** | 田中 陽介 |
| **年齢** | 30歳 |
| **職業** | Webエンジニア |
| **技術レベル** | 高（Markdown記法に精通、CLI操作に慣れている） |

### 背景
- フルスタックエンジニアとして働きながら、技術ブログを定期的に執筆
- 主にZennで技術記事を公開し、microCMSで個人サイトのコンテンツを管理
- 現在はVS Codeで記事を書いてから手動でコピー＆ペーストしている

### 目標
- 技術記事を素早く執筆し、Zennに直接投稿したい
- コードブロック、数式、Mermaidダイアグラムを快適に書きたい
- 記事の管理と投稿を一つのツールで完結させたい

### ペインポイント
- VS Codeでは記事のプレビューが投稿先と一致しない
- 手動でのコピー＆ペーストが面倒で、画像の再アップロードも手間
- 複数プラットフォームへの投稿時にフォーマットの変換が必要
- Zenn独自の記法（メッセージ、アコーディオン等）のプレビューができない

### 利用シナリオ
1. アプリを起動し、ワークスペースから新しい技術記事を作成
2. WYSIWYGモードでコードブロックやダイアグラムを含む記事を執筆
3. Zennプレビューモードで投稿先での見た目を確認
4. エディターから直接Zennに下書き投稿
5. Zenn上で最終確認後、公開

## ペルソナ2: 佐藤 美咲（一般ブロガー）

| 項目 | 内容 |
|---|---|
| **名前** | 佐藤 美咲 |
| **年齢** | 27歳 |
| **職業** | フリーランスライター |
| **技術レベル** | 中（Markdownの基本は知っているが、複雑な記法は苦手） |

### 背景
- フリーランスのライターとして複数のメディアに記事を寄稿
- noteで個人ブログを運営し、企業のmicroCMSベースのオウンドメディアにも寄稿
- 現在はnoteのエディターとGoogleドキュメントを併用している

### 目標
- 一つのエディターで書いた記事を複数のプラットフォームに投稿したい
- Markdownの記法を意識せずに、直感的にリッチなコンテンツを作成したい
- 画像の挿入と管理を簡単にしたい

### ペインポイント
- プラットフォームごとにエディターが異なり、学習コストが高い
- 画像のアップロードを毎回手動で行う必要がある
- 記事の下書きが複数ツールに分散して管理しにくい
- noteのリッチエディターは便利だがオフラインで使えない

### 利用シナリオ
1. アプリを起動し、ワークスペースから新しい記事を作成
2. WYSIWYGモードで文章を書き、ドラッグ&ドロップで画像を挿入
3. 見出し、リスト、引用などを視覚的に編集
4. 記事完成後、noteに直接投稿（画像も自動アップロード）
5. 同じ記事をmicroCMS向けにエクスポートして企業メディアにも寄稿

## ペルソナ-ストーリーマッピング概要

| エピック | 田中（テック） | 佐藤（一般） |
|---|---|---|
| コアエディター | コードブロック、数式、ダイアグラム重視 | 直感的な文章編集、ドラッグ&ドロップ重視 |
| ファイル管理 | 技術記事のカテゴリ管理 | 複数メディア向け記事の一元管理 |
| プラットフォーム連携 | Zenn直接投稿、microCMS API連携 | note直接投稿、microCMSエクスポート |
| 画像管理 | ダイアグラム画像の自動生成・アップロード | ドラッグ&ドロップ画像挿入・自動アップロード |
| プラグイン | Zenn独自記法、Mermaid対応 | 最小限（デフォルト機能で十分） |
| 設定 | キーバインドカスタマイズ、テーマ | フォントサイズ、表示設定 |
